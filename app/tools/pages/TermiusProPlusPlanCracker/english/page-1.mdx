<p className="text-5xl mt-8" style={{ fontVariant: "small-caps" }}>Steps</p>

<hr style={{ margin: "10px 0", border: "none", borderTop: "1px solid #ffffff" }} />

First, ensure that any paid plan is not activated.

<br/>![Home](pages/TermiusProPlusPlanCracker/english/images/1.png) <br/>

Click <i>Hamburger Menu</i> on top left, and click <i>"Settings"</i>.

<br/>![Home](pages/TermiusProPlusPlanCracker/english/images/2.png) <br/>

As you can see, current plan is <i>Trial</i>, so let activate paid plan. <br/>
Close termius, and then press "Win + r", copy and paste this:

<div className="block text-center my-4 copyable">
    %LocalAppData%\Programs\Termius\resources
</div>

This will open a explorer, type <i>"cmd"</i> on upper box.

<br/>![Home](pages/TermiusProPlusPlanCracker/english/images/3.png) <br/>

Press enter and <i>command prompt</i> will appear. <br/>
Run following commands:

<div className="block text-center my-4 copyable">
    asar extract app.asar app.asar.unpacked.own <br/>
    xcopy /E /I /Y app.asar.unpacked app.asar.unpacked.own <br/>
    ren app.asar app-original.asar <br/>
    del app-update.yml <br/>
</div>

Back to <i>file explorer</i> again.

<br/>![Home](pages/TermiusProPlusPlanCracker/english/images/4.png) <br/>

Open <i>"app.asar.unpacked.own -> background-process -> assets"</i>. <br/>
There is a file named <i>"main-*.js"</i> (\* is random string) on current folder, open that with <i>notepad</i>.

<br/>![Home](pages/TermiusProPlusPlanCracker/english/images/5.png) <br/>

Search and replace 

<div className="block text-center my-4 copyable">
    bulkAccount()\{return this.get(_t.bulkAccount)\}
</div>

with

    <span className="absolute -top-3 left-4 bg-black px-2 text-white">
        Pro plan
    </span>

    <span className="absolute -top-3 left-4 bg-black px-2 text-white">
        Business plan
    </span>

Save and close, then back to command prompt again. <br/>
Run following command:

<div className="block text-center my-4 copyable">
    asar pack app.asar.unpacked.own app.asar --unpack-dir "\{node_modules,out\}" --exclude-hidden
</div>

Open termius and you got the plan that you chose.
